## 🚀 快速开始（换电脑必看）

1. 克隆仓库：`git clone https://github.com/solomon0256/solomon0256.github.io.git`
2. 安装环境：参考 [01-环境配置.md](./01-环境配置.md)
3. 本地预览：`bundle exec jekyll serve`
4. 开始开发！
